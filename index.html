Portfolio - Aubrain Virtual

site is hosted on GitHub: https://au-brain.github.io/

1. The web application is validated through validator w3 with few errors that can't be fixed. Also tested for web accessibility with no major errors.
Here is the error from the validator that can't be fixed...
[Error: Element img not allowed as child of element ul in this context. (Suppressing further errors from this subtree.)]

2. The site is responsive at a Media Query max-width of 640px. Elements display as blocks. Added Javascript to hide and unhide the Navbar Menu on a Smaller screen view.

3. The site is styled with plain CSS. No templated wireframe or bootstrap was used. Layout, animation, hovering, etc. done with CSS.

4. There are at least 4 images incorporated each on the Home Page and About Page.
